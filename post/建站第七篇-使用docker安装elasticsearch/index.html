Docker elasticSearch 建站

建站第七篇 使用Docker安装elasticSearch

Posted by     "DD" on Wednesday, August 7, 2019

在安装好docker之后，部署docker版的应用，只需要一句代码就好。

1 环境

os：ubuntu 16.04 docker版本：Docker CE，Docker version 19.03.1

2 安装es

docker run -it -p 9200:9200 -p 9300:9300 --name es elasticsearch:6.4.0

报了一个错

[1]: max virtual memory areas vm.max_map_count [65530] is too low, increase to at least [262144]

需要修改/etc/sysctl.conf

$ sudo vim /etc/sysctl.conf

# 添加
vm.max_map_count=262144

$ sudo sysctl -p

再次启动

docker start es

部署完成。

3 访问

访问 http://localhost:9200,看到这样的图，就表示安装成功了。
